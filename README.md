# Business Formation Services

A modern web application for LLC formation services across all 50 US states. Built with Next.js, featuring a complete checkout flow, dynamic state-based pricing, and integrated payment processing.

## Features

- **All 50 States** - LLC formation services available nationwide with state-specific filing fees
- **Dynamic Pricing** - Real-time pricing based on selected state and services
- **Rush Processing** - Optional expedited filing where available
- **Registered Agent Services** - Professional registered agent service included
- **Secure Checkout** - Multi-step checkout with Stripe payment integration
- **User Dashboard** - Track order status and manage business documents
- **OAuth Authentication** - Google sign-in with session management
- **Internationalization** - Multi-language support (English/Spanish)

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: Custom auth with OAuth support
- **Payments**: Stripe
- **Styling**: Tailwind CSS v4
- **Deployment**: Vercel
